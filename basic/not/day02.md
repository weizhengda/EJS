## EJS

###标签
1. 用<%...%>包含js代码
2. 用<%=...%>输出变量 变量若包含 '<' '>' '&'等字符 会被转义
3. 用<%-...%>输出变量 不转义
4. 用<%- include('user/show') %>引入其他模板 包含 ./user/show.ejs
5. 用<%# some comments %>来注释，不执行不输出
6. <%% 转义为 '<%'
7. <% ... -%> 删除新的空白行模式?
8. <%_ ... _%> 删除空白符模式

###布局
1.EJS 不会特别地支持区块，但是可以采用包含头部和尾部的方法来实现局部，像这样：
 
 ```
 <%- include('header') -%>
 <h1>
   Title
 </h1>
 <p>
   My page
 </p>
 <%- include('footer') -%>
 ```
###循环结构
1. if循环

```
  <% if (user) { %>
    <h2><%= user.name %></h2>
  <% } %
```
